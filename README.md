# 🌦️ Weather App

A modern and responsive weather application built using **HTML**, **CSS**, and **JavaScript**. This application provides real-time weather information, location-based weather updates, and a 5-day weather forecast using the **Open-Meteo API**. It features dynamic weather backgrounds, animated weather icons, and a clean, responsive user interface.
---
## 🚀 Live Demo
🔗 **Live Website:**
https://weather-app-ruddy-rho-64.vercel.app
---
## ✨ Features
* 🌍 Search weather by city name
* 📍 Get weather using your current location
* 🌡️ Toggle between Celsius (°C) and Fahrenheit (°F)
* 📅 5-Day weather forecast
* 🎨 Dynamic weather backgrounds based on weather conditions
* ☀️ Animated weather icons
* ⚡ Loading animation while fetching weather data
* 📱 Fully responsive design
* ❌ User-friendly error handling for invalid locations
---
## 🛠️ Technologies Used
* HTML5

* Open-Meteo API

